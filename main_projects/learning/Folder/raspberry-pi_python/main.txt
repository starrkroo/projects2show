В современном мире преобладает минимализм, который по сути облегчает жизнь. В технологическом мире минимализм проявляет себя с большой буквы. Использование объемных, массивных конструкций не приемлемо. Перенос данных с одного девайся на другой - также усложняет процесс. Этому вопросу запросто ответит такой компьютер, как Raspberry Pi

Raspberry Pi  - одноплатный компьютер размером с банковскую карту, изначально разработанный как бюджетная система для обучения информатике, но позже получивший более широкое применение и известность. 
Данный одноплатный компьютер обладает всеми качествами настоящего компьютера. На нем можно переносить данные, выводить картинку на экран через кабель HDMI. Подключить к интернету через ETHERNET кабель. И это еще не предел. К данном компьютеру можно подключить разного рода аппаратуру: мышь, клавиатуру, наушники, колонки и так далее. На нем вы можете без всяких обдумываний транспортировать данные, так как этот одноплатный компьютер уместится даже в карман. 

Из начинки у него
	~ Процессор: Arm cortex-a53 1.2 ГГц (64-битный 4 ядерный)
	~ Видео GPU: двухъядерный VideoCore 4
	~ Оперативная память 1 Гб(на обратной стороне малинки)
	~ Вайфай и bluetooth в одном чипе - broadcom bcm 43438
	~ 4 выхода usb 2.0
	~ Сетевой адаптер Ethernet скорость работы 100 мегабит
	~ За их работу отвечает usb хаб LAN9514-jzx
	~ Для подключения наушников или колонок - jack 3.5 мм также является композитным rca-видеовыходом и может использоваться для вывода видео в форматах PAL NTSC
	~ Hdmi выход
	~ Microusb для подключения питания(требуется для нормального питания 2.5 Ампера )
	~ Два светодиода показывают на наличие питания(красный) и обращение к sd карте
	~ {
			Также распаяны два разъема для подключения
			csi-2(camera serial interface) - для подключения камеры(снизу относительно usb)
			Dsi(display serial interface) - для подключения дисплея (сверху относительно us
		}
	~ Рядом с dsi антенна контролёра беспроводной связи 
	~ На лицевой стороне 40-пиновый разъем GPIO(general purpose input/output) - подключение внешних устройвст для ввода и вывода

Raspberry Pi Может служить не только как компьютер, на нем можно выполнять разные трюки используя 40-пиновый разъем GPIO(general purpose input/output)
Далее я прикреплю простые и элементарные коды для получения пояснения как и что работает:
1) 	

			#!/usr/bin/env python3

			import RPi.GPIO as gpio

			# запускаем наш боард
			gpio.setmode(gpio.BOARD)

			# говорим конкретному порту для подачи тока
			# gpio.setup(number_of_the_port, what_is_he_doing)
			gpio.setup(7, gpio.out) # gpio.in

			# включение диода
			# gpio.output(number_of_the_port, true=1, false=0)
			gpio.output(7, 1)

			# выключение
			gpio.output(7, 0)

			# следует делать при окончании работы 
			gpio.cleanup()


2)  
			# gpio.input  - pressed button(checks)
			# gpio.output - outputs to start diod working

			import RPi.GPIO as gpio
			gpio.setmode(gpio.BOARD)
			# here is the diode
			gpio.setup(7, gpio.OUT)

			# here is the button
			gpio.setup(3, gpio.IN)


			while True:
			  # false - if pressed
			  # true  - if not pressed
			  if gpio.input(3) == False:
			    gpio.out(7, 1)
			  else:
			    gpio.output(7, 0)


3) 

			import RPi.GPIO as gpio
			gpio.setmode(gpio.BOARD)
			gpio.setup(7, gpio.OUT)

			while True:
			  if len(input()) == 0:
			    gpio.output(7, 1)
			  else:
			    break

			    gpio.output(7, 0)
